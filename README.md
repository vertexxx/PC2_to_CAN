# PC2_to_CAN
Using the CREST2 rest api to transfert "Project Cars 2" game data to CAN (via PEAK Systems API).

## Usage
1. Run Game "Project Cars 2" and start driving
2. Make sure PCAN USB adapter is connected
3. Run CREST2.exe
4. Run python script
